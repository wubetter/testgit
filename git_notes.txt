1.创建版本库 并初始化
mkdir testgit
git init

2.文件添加 到缓存区中
git add readme.txt 
3.查看文件状态
git status
注意  git文件状态分3中  修改过的未推进缓存区（红色提示）  修改过的 推进缓存区（绿色显示） 已经提交过

4.提交
git commit -m '本次修改的注释'

5.比较文件差异性
git diff readme.txt

git log   //显示commit id ,作者, 更新日期,  更新描述
